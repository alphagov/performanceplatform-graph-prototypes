# Investigation into graphing libraries for Performance Platform

- [C3](http://c3js.org)
- [NVD3](http://nvd3.org)

## To install
```
npm install
```

## To run

```
node server
```

Hit

http://localhost:5000/performance/<dashboard-slug>?lib=<graph-library>

eg. -

http://localhost:5000/performance/prison-visits?lib=c3
